# install
**install on parrot,ubuntu and kali:**

```
$ apt-get install git
$ git clone https://github.com/Ranginang67/sqlku
$ cd sqlku
$ chmod +x setup.sh
$ ./setup.sh
$ perl sqlku.pl
```

**install on Termux:**

```
$ pkg install git
$ git clone https://github.com/Ranginang67/sqlku
$ cd sqlku
$ chmod +x setup.sh
$ ./setup.sh
$ perl sqlku.pl
```
